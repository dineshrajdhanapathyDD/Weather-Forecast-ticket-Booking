**Video Structure (< 5 minutes)**:

   - "I built Weather-Wise Flight Booking Agent"
   - "It combines weather intelligence with fare trends"

2. **Feature Overview (2 minutes)**
   - Show landing page
   - Explain dual interface
   - Highlight key features

3. **Search Mode Demo (1 minute)**
   - Enter route (JFK to LAX)
   - Select dates
   - Show recommendation results
   - Explain weather risk and fare trend

4. **Chat Mode Demo (1 minute)**
   - Ask natural language question
   - Show AI response
   - Demonstrate conversation flow

5. **Conclusion (30 seconds)**
   - Recap benefits
   - Mention Amazon Nova
   - Thank viewers

**Recording Tools**:
- **Windows**: OBS Studio (free) or Xbox Game Bar (built-in)
- **Mac**: QuickTime Player (built-in) or OBS Studio
- **Linux**: OBS Studio or SimpleScreenRecorder

**Recording Settings**:
- Resolution: 1080p (1920x1080)
- Frame rate: 30 fps
- Audio: Clear microphone
- Length: 4-5 minutes max
